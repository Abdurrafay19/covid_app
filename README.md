# 🦠 Flutter COVID-19 Tracker App

A cross-platform mobile application built with **Flutter** that tracks global and country-wise COVID-19 statistics in real time.  
The app presents updated data on confirmed cases, recoveries, and deaths through interactive charts and clean visuals.

---

## 🌍 Overview

The Flutter COVID-19 Tracker App collects verified data from reliable sources.  
It provides users with an easy-to-read dashboard and graphical trends to monitor the pandemic’s progress globally and regionally.

---

## ✨ Key Features

- Live global and country-specific COVID-19 statistics  
- Graphs for daily and total cases, recoveries, and deaths  
- Search bar to find any country’s data  
- World map or list view to visualize affected regions  
- Real-time data refresh with pull-to-refresh support  
- Clean typography and color-coded indicators  

---

## 🧠 Tech Stack

- **Framework:** Flutter (Dart)  
- **API Provider:** disease.sh / COVID19API / WHO open data  
- **State Management:** Provider 
- **Networking:** `http`
- **Charts:** `fl_chart`
